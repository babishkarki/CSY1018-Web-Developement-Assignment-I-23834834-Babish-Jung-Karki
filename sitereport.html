Report

Introduction:

The website is created as a personal showcase for a web development project. The website is hosted on GitHub Pages and features a straightforward layout with five sections—Home, Portfolio, Experience, Report and Contact—with a navigation menu. HTML and CSS are used in the development of the website.

Design:

The design was created on a figma wireframe and then applied using html and css. The website features a simple design with a white body background and whiite color navigation bar with box shadow and white color text for logo and navigation menu sections with gol color hover. The website is simple to read because the font size is uniform throughout. The website's top bar contains a navigation menu, and selecting any of the items takes the user to the appropriate section. Because of its responsive design, which adjusts to various screen sizes, it is simple to see the website on many devices.

Coding:

The code is well-structured, and the HTML and CSS are separated into different files, making it easy to read and maintain it is well-commented, making it easy to understand as it follows the best practices for web development. An anchor tag providing each specific files is used to provide smooth changing of a page when a user clicks on a navigation bar item, grid layout on CSS to align page elements based on sequenced columns and rows.

Debugging:

Dealing with errors and bugs was among the biggest difficulties. Yet, debugging was a crucial step in the coding process, and mistakes offered chances to learn new knowledge and develop technical skills.

Module Experience:

Throughout the module, it was possible to gain practical experience in designing and building a simple web page , as well as an understanding of the underlying technologies and programming languages involved. This module also challenged to think creatively and outside the box when it came to designing and developing websites. It encouraged to explore different design techniques and approaches, which helped to develop own unique style and perspective.

Conclusion:

Overall, learning about the principles of web development, programming, design, and debugging methods proved to be a rewarding experience. Although it was challenging at times, The experience resulted in significant gains. Individual abilities was discovered, and there is a desire to keep moving forward in this field. As abilities and knowledge in web development continue to grow, the satisfaction of having constructed a structure through this module is fulfilling.
