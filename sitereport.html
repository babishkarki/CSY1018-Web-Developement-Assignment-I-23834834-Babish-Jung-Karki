Report

The website is created as a personal showcase for a web development project. The website is hosted on GitHub Pages and features a straightforward layout with five sections—Home, Portfolio, Experience, Report and Contact—with a navigation menu. HTML and CSS are used in the development of the website.

The website features a simple design with a white background and a black font color. The website is simple to read because the font size is uniform throughout. The website's top bar contains a navigation menu, and selecting any of the items takes the user to the appropriate section. Because of its responsive design, which adjusts to various screen sizes, it is simple to see the website on many devices.

The website's code quality is good, and it follows the best practices for web development. The HTML and CSS are separated into separate files and the code is nicely organized, making it simple to understand and update. The CSS code is used to enable smooth scrolling when a user clicks on a menu item, and it is well-commented, making it easy to understand.

The website is overall well-designed, navigable, and the content is well-structured and simple to comprehend. The website's code is of good quality and follows the best practice for web development.
